# دليل البدء السريع

## 🚀 بدء الاستخدام

### 1. أضف منتجاتك أولاً
```
1. اذهب إلى: المخزون
2. اضغط: إضافة منتج جديد
3. أدخل: اسم المنتج، السعر، الفئة، الكمية
4. اضغط: حفظ
```

### 2. سجل عمليات الشراء (اختياري)
```
1. اذهب إلى: المشتريات
2. اضغط: عملية شراء جديدة
3. أدخل: اسم المورد والمنتجات والكميات
4. اضغط: حفظ عملية الشراء
→ المخزون يُحدّث تلقائياً ✓
```

### 3. ابدأ البيع
```
1. اذهب إلى: نقطة البيع
2. ابحث عن المنتج أو اختره
3. أدخل الكمية المطلوبة
4. اختر طريقة الدفع
5. اضغط: إتمام العملية
→ الفاتورة تُنشأ تلقائياً ✓
```

## 📊 الصفحات الرئيسية

### لوحة التحكم (Dashboard)
**ماذا تجد هنا:**
- إجمالي المبيعات اليومية
- عدد الفواتير
- المنتجات منخفضة المخزون
- آخر الفواتير

### نقطة البيع (POS)
**الميزات:**
- بحث فوري عن المنتجات
- سلة المشتريات
- حساب الضرائب تلقائياً
- إنشاء فاتورة

### المخزون (Inventory)
**الإجراءات:**
- إضافة/تعديل/حذف منتجات
- تتبع الكميات
- تنبيهات المخزون المنخفض

### المشتريات (Purchases) ⭐
**تسجيل:**
- شراء من الموردين
- تحديث المخزون تلقائياً
- متابعة تاريخ كل عملية

### الإنتاج (Production) ⭐
**تتبع:**
- عمليات الإنتاج
- حالة الإنتاج (معلق/مكتمل/ملغى)
- تحديث المخزون عند الإتمام

### الاستهلاك (Consumption) ⭐
**تسجيل:**
- الكميات المستهلكة أو التالفة
- أسباب محددة (تلف، عينة، إلخ)
- تقليل المخزون تلقائياً

### الفواتير (Invoices)
**إدارة:**
- عرض جميع الفواتير
- البحث والتصفية
- تصدير CSV

### التقارير (Reports)
**التحليل:**
- رسوم بيانية تفاعلية
- الإيرادات والأرباح
- الإحصائيات اليومية

---

## 💡 نصائح مهمة

1. **البحث السريع**: في كل صفحة يوجد شريط بحث أعلى
2. **التصفية**: استخدم الفلاتر لتضييق البحث
3. **الشبكة**: جميع الصفحات تتكيف مع حجم الشاشة
4. **الحفظ**: البيانات تُحفظ تلقائياً، لا تقلق!
5. **الحذف**: يمكن حذف أي عملية إذا لزم الأمر

---

## 🔄 دورة العمل المثالية

```
الصباح:
  1. راجع لوحة التحكم
  2. تحقق من المخزون
  3. ابدأ البيع

أثناء اليوم:
  4. استخدم نقطة البيع للبيع
  5. سجل أي استهلاك أو تلف
  6. حدّث الإنتاج إن وجد

المساء:
  7. راجع الفواتير
  8. تحقق من التقارير
  9. احسب الأرباح
```

## ❓ أسئلة شائعة

**س: هل ستُفقد البيانات عند إغلاق التطبيق؟**
ج: لا، البيانات محفوظة محلياً وآمنة

**س: كيف أصدر تقرير؟**
ج: اذهب إلى التقارير وستجد جميع الإحصائيات

**س: هل يمكن تعديل فاتورة؟**
ج: الفواتير تُنشأ وتُحفظ كما هي، لكن يمكنك حذفها

**س: كيف أحذف منتج؟**
ج: اذهب إلى المخزون واضغط الحذف على المنتج

**س: ماذا لو أردت تصدير البيانات؟**
ج: في الفواتير يوجد زر لتصدير CSV
