# 🛍️ نظام POS الشامل - Zazom

## مرحباً بك!

تم إعادة بناء المشروع بنجاح وتحويله إلى **نظام POS محترف** متكامل مع تصميم عصري وواجهة سهلة الاستخدام.

---

## 📋 الهيكل الجديد

```
app/
├── components/
│   ├── App.tsx              # المكون الرئيسي (التنقل)
│   ├── Dashboard.tsx        # لوحة التحكم
│   ├── POS.tsx             # نقطة البيع
│   ├── Inventory.tsx       # إدارة المخزون
│   ├── Invoices.tsx        # الفواتير
│   └── Reports.tsx         # التقارير
├── store/
│   ├── StoreContext.tsx    # إدارة الحالة (Context API)
│   └── types.ts            # أنواع البيانات
├── hooks/                  # هوكس مخصصة
├── lib/                    # دوال مساعدة
├── layout.tsx              # تخطيط الصفحة
├── page.tsx               # نقطة الدخول
└── globals.css            # الأنماط الكلية
```

---

## 🚀 كيفية الاستخدام

### البدء
```bash
pnpm dev
```
افتح [http://localhost:3000](http://localhost:3000)

### الميزات الأساسية

#### 1️⃣ **لوحة التحكم**
- نظرة عامة سريعة على الأداء
- إحصائيات فورية
- تنبيهات المخزون
- آخر الفواتير

#### 2️⃣ **نقطة البيع**
- واجهة سريعة للبيع
- بحث فوري عن المنتجات
- حساب الضرائب تلقائياً
- طرق دفع متعددة

#### 3️⃣ **إدارة المخزون**
- إضافة/تعديل/حذف المنتجات
- تحديث الكميات
- تنبيهات المخزون المنخفض

#### 4️⃣ **الفواتير**
- عرض جميع الفواتير
- تصفية حسب الحالة
- طباعة وتصدير CSV

#### 5️⃣ **التقارير**
- رسوم بيانية فورية
- تحليل الإيرادات والأرباح
- ملخص يومي تفصيلي

## 💾 إدارة البيانات

**البيانات محفوظة محلياً في localStorage**:
- `pos_products` - المنتجات
- `pos_invoices` - الفواتير
- `pos_reports` - التقارير اليومية

## 🔧 التخصيص

### تغيير الألوان
عدّل في `app/components/App.tsx` و `app/globals.css`

### إضافة منتجات
استخدم لوحة "المخزون" من الواجهة مباشرة

### تغيير نسبة الضريبة
عدّل في `app/components/POS.tsx` السطر:
```typescript
const [taxPercent, setTaxPercent] = useState(14)
```
