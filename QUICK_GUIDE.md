# دليل الاستخدام السريع

## 🚀 البدء الفوري

```bash
# تشغيل التطبيق
pnpm dev

# ثم افتح:
http://localhost:3000
```

---

## 📊 الصفحات الرئيسية

### 1. لوحة التحكم (Dashboard)
- عرض ملخص البيانات الكامل
- إحصائيات المبيعات
- ملخص بيانات حي
- روابط سريعة

### 2. نقطة البيع (POS)
```
1. ابحث عن المنتج
2. أضف الكمية
3. انقر "إضافة"
4. كرر للمنتجات الأخرى
5. انقر "الدفع"
```

### 3. الحاسبة (Calculator)
- حاسبة ذكية كاملة
- اختر المنتج
- ادخل الكمية
- أضف للسلة

### 4. المخزون (Inventory)
- عرض جميع المنتجات
- إضافة منتجات جديدة
- تحديث الكميات
- حذف منتجات

### 5. المشتريات (Purchases)
- تسجيل الشراء من الموردين
- إضافة المنتجات
- تحديث المخزون
- حفظ الملاحظات

### 6. الإنتاج (Production)
- تسجيل عمليات الإنتاج
- تتبع الحالة
- إضافة الملاحظات

### 7. الاستهلاك (Consumption)
- تسجيل المنتجات التالفة
- تحديد السبب
- تقليل المخزون

### 8. الفواتير (Invoices)
- عرض جميع الفواتير
- تتبع حالة الدفع
- طباعة الفواتير

### 9. اكسل (Excel)
```
تصدير:
1. اذهب إلى صفحة "اكسل"
2. اضغط "تصدير"
3. سيتم تحميل الملف

استيراد:
1. اضغط "استيراد"
2. اختر ملف Excel
3. سيتم الاستيراد تلقائياً
```

### 10. التقارير (Reports)
- رسوم بيانية
- إحصائيات يومية
- تحليل المبيعات

### 11. الإعدادات (Settings)
```
معلومات العملة:
- الرمز: ج.م
- الاسم: الجنيه المصري
- ضريبة: 14%

إحصائيات البيانات:
- عدد المنتجات
- عدد الفواتير
- إجمالي البيانات

حذف البيانات:
1. انقر "حذف جميع البيانات"
2. أكد الحذف
3. سيتم حذف كل شيء
```

## 💰 الحسابات المالية

### ضريبة القيمة المضافة
```
السعر الأساسي: 100 ج.م
الضريبة (14%): 14 ج.م
─────────────────────────
الإجمالي: 114 ج.م
```

### الفاتورة
```
منتجات:
- ماء معدني (5 × 5 ج.م) = 25 ج.م
- عصير (3 × 12 ج.م) = 36 ج.م
─────────────────────────
الإجمالي الجزئي: 61 ج.م
الضريبة (14%): 8.54 ج.م
─────────────────────────
الإجمالي النهائي: 69.54 ج.م
```

## 🔔 الإشعارات

### أنواع الإشعارات:
- **أخضر** ✓ - نجاح العملية
- **أحمر** ✗ - خطأ أو تحذير
- **أصفر** ⚠️ - تنبيه
- **أزرق** ℹ️ - معلومة

### الإشعارات الشائعة:
- "تم إضافة المنتج بنجاح"
- "تم إنشاء فاتورة جديدة"
- "تم تحديث المخزون"
- "السلة فارغة"

## 💾 الحفظ الآلي

البيانات تُحفظ تلقائياً عند:
- إضافة منتج
- إنشاء فاتورة
- إضافة شراء
- تحديث مخزون
- إضافة إنتاج
- تسجيل استهلاك

**لا تحتاج للضغط على "حفظ"**

---

## 🆘 استكشاف الأخطاء

### السلة فارغة
**الحل**: أضف منتجات قبل محاولة الدفع

### لا توجد منتجات
**الحل**: اذهب للمخزون وأضف منتجات

### الكمية غير متوفرة
**الحل**: الكمية في المخزون أقل من المطلوبة

### رسالة خطأ
**الحل**: جرب تحديث الصفحة أو مسح بيانات المتصفح
