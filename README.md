# Zazom - نظام إدارة المخزون المتقدم

## نظرة عامة
Zazom هو تطبيق ويب احترافي لإدارة المخزون والعمليات الحسابية المتقدمة. مصمم بتصميم حديث بأسلوب Glassmorphism مع دعم كامل للغة العربية.

## المميزات الرئيسية

### 1. لوحة التحكم (Dashboard)
- عرض إجمالي قيمة المخزون الحالية
- عدد العمليات اليومية
- عدد المنتجات بمخزون منخفض
- إجمالي عدد المنتجات
- رسم بياني لاتجاه المبيعات (آخر 7 أيام)

### 2. إدارة المخزون (Inventory Management)
- إضافة منتجات جديدة بسهولة
- تعديل بيانات المنتجات (الكمية والسعر)
- حذف المنتجات غير المستخدمة
- البحث السريع عن المنتجات
- جدول مفصل لجميع المنتجات

### 3. الحاسبة المتقدمة (Calculator)
- حاسبة علمية كاملة
- حفظ العمليات الحسابية تلقائياً إلى السجل
- تحديد نوع العملية (إدخال/إخراج)
- تتبع تفصيلي لجميع العمليات

### 4. سجل العمليات (Operation Log)
- عرض جميع العمليات بترتيب زمني
- حذف العمليات غير المطلوبة
- تصدير البيانات إلى CSV
- معلومات تفصيلية عن كل عملية

### 5. الوضع المظلم (Dark Mode)
- تبديل سهل بين الوضع الفاتح والمظلم
- حفظ التفضيلات تلقائياً
- راحة العين عند الاستخدام الطويل

## البيانات التقنية

### الكومبونينتات المستخدمة:
- **React 19.2** - مكتبة المواجهات
- **Next.js 16** - إطار العمل
- **Tailwind CSS 4** - نمط التصميم
- **Shadcn UI** - مكتبة المكونات
- **Recharts** - الرسوم البيانية
- **Lucide React** - الأيقونات
- **next-themes** - إدارة المواضيع

### تخزين البيانات:
البيانات يتم حفظها محلياً في `localStorage` للبساطة. يمكن لاحقاً ربطها بقاعدة بيانات مثل Firebase أو Supabase.

## التثبيت والتشغيل

```bash
# تثبيت المتطلبات
pnpm install

# تشغيل خادم التطوير
pnpm dev

# بناء النسخة الإنتاجية
pnpm build

# تشغيل النسخة الإنتاجية
pnpm start
```
