## نظام إدارة نقاط البيع (POS) - Zazom

نظام شامل ومتكامل لإدارة نقاط البيع والفواتير والمخزون بتصميم عصري احترافي مع دعم كامل للعربية.

### 🎯 المميزات الرئيسية

#### 1. **لوحة التحكم (Dashboard)**
- عرض ملخص شامل للأداء اليومي
- إحصائيات فورية للإيرادات والأرباح
- تنبيهات المخزون المنخفض
- روابط سريعة للأقسام الأخرى
- آخر الفواتير بمعاينة فورية

#### 2. **نقطة البيع (POS)**
- واجهة سهلة وسريعة للبيع
- بحث فوري عن المنتجات
- إضافة/حذف العناصر من السلة بسهولة
- حساب الضريبة تلقائياً (14% افتراضياً)
- خيارات دفع متعددة (نقد، بطاقة، تحويل)
- إمكانية إضافة اسم العميل
- حساب الإجمالي مباشرة

#### 3. **إدارة المخزون (Inventory)**
- عرض جدول كامل للمنتجات
- إضافة/تعديل/حذف المنتجات
- تحديث الكميات والأسعار
- تنبيهات للمخزون المنخفض (أقل من 10 وحدات)
- دعم الباركود (اختياري)

#### 4. **إدارة الفواتير (Invoices)**
- عرض جميع الفواتير المُنشأة
- تصفية حسب الحالة (مدفوعة/غير مدفوعة)
- طباعة الفواتير
- تحديد الفاتورة كمدفوعة
- تصدير البيانات إلى CSV
- عرض تفاصيل العناصر لكل فاتورة

#### 5. **التقارير والإحصائيات (Reports)**
- رسوم بيانية للإيرادات والأرباح
- اتجاهات النفقات
- ملخص يومي تفصيلي
- إجماليات الفترة
- بيانات آخر 7 أيام في الرسوم البيانية

### 🏗️ البنية التقنية

#### المكتبات المستخدمة:
- **React 19.2** - مكتبة واجهات المستخدم
- **Next.js 16** - إطار عمل React متقدم
- **Tailwind CSS 4** - أنماط CSS حديثة
- **Recharts** - رسوم بيانية تفاعلية
- **Lucide React** - أيقونات احترافية
- **TypeScript** - لغة برمجية آمنة النوع

#### هيكل المشروع:
```
src/
├── components/
│   ├── App.tsx           # المكون الرئيسي
│   ├── Dashboard.tsx     # لوحة التحكم
│   ├── POS.tsx          # نقطة البيع
│   ├── Inventory.tsx    # إدارة المخزون
│   ├── Invoices.tsx     # إدارة الفواتير
│   └── Reports.tsx      # التقارير
├── store/
│   ├── StoreContext.tsx # إدارة الحالة
│   └── types.ts         # أنواع البيانات
├── layout.tsx           # تخطيط الصفحة
├── page.tsx             # الصفحة الرئيسية
└── globals.css          # الأنماط العامة
```

### 🎨 التصميم والواجهة

**نمط Glassmorphism**
- خلفيات شفافة مع تأثير Blur
- ألوان احترافية ومتناسقة
- واجهة داكنة مع خيار الوضع الفاتح
- دعم كامل لـ RTL (العربية)
- تصميم مستجيب يعمل على جميع الأجهزة

**لوحة الألوان**:
- الأساسي: أزرق سماوي (#3b82f6)
- الثانوي: أحمر برتقالي (#ea580c)
- الدعم: أخضر (#10b981)
- النص: أبيض مع تدرجات رمادية

### 💾 إدارة البيانات

يتم حفظ جميع البيانات محلياً في `localStorage`:
- `pos_products` - المنتجات
- `pos_invoices` - الفواتير
- `pos_reports` - التقارير اليومية

### 📊 حالات الاستخدام

1. **بيع المنتجات** - استخدم نقطة البيع لإتمام العملية
2. **تتبع المخزون** - أضف منتجات جديدة وحدّث الكميات
3. **متابعة المبيعات** - عرض وطباعة الفواتير
4. **تحليل الأداء** - ادرس التقارير والإحصائيات

### 🔧 ميزات متقدمة

- **معادلة الضريبة الديناميكية** - غيّر نسبة الضريبة حسب الحاجة
- **تصفية الفواتير** - ابحث حسب الحالة (مدفوعة/معلقة)
- **تصدير البيانات** - احفظ الفواتير كـ CSV
- **تحديثات فورية** - جميع البيانات تُحدّث فوراً
- **تنبيهات ذكية** - نبّه عندما ينخفض المخزون
