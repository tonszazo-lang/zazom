# نظام POS المتقدم - دليل الميزات

## نظرة عامة

نظام إدارة نقاط البيع المتقدم يوفر حلاً شاملاً لإدارة المبيعات والمخزون والإنتاج والمشتريات.

---

## 📊 المميزات الرئيسية

### 1. لوحة التحكم (Dashboard)
**الوصول السريع للإحصائيات الهامة:**
- إجمالي المبيعات اليومية
- عدد الفواتير
- التنبيهات (مخزون منخفض)
- آخر الفواتير المضافة
- رسم بياني لاتجاهات المبيعات

### 2. نقطة البيع (POS)
**واجهة سهلة للبيع السريع:**
- بحث فوري عن المنتجات
- إضافة المنتجات للسلة
- حساب تلقائي للضرائب
- طرق دفع متعددة
- إنشاء فواتير على الفور
- طباعة الفواتير

### 3. إدارة المخزون (Inventory)
**التحكم الكامل بالمنتجات:**
- إضافة منتجات جديدة
- تعديل الأسعار والكميات
- حذف المنتجات
- تنبيهات المخزون المنخفض
- التصنيفات والفئات
- البحث والتصفية السريعة

### 4. المشتريات (Purchases) ⭐ **جديد**
**إدارة عمليات الشراء من الموردين:**
- تسجيل عمليات شراء جديدة
- تحديد المورد والمنتجات والكميات
- حساب التكلفة الإجمالية تلقائياً
- تحديث كمية المخزون تلقائياً
- تتبع تاريخ كل عملية شراء
- إضافة ملاحظات للعملية
- عرض شبكي لجميع المشتريات
- حذف العمليات إذا لزم الأمر

### 5. الإنتاج (Production) ⭐ **جديد**
**تتبع عمليات الإنتاج:**
- تسجيل عمليات إنتاج جديدة
- تحديد المنتج والكمية المنتجة
- تتبع حالة الإنتاج (قيد الانتظار / مكتمل / ملغى)
- تحديث تلقائي للمخزون عند إتمام الإنتاج
- إضافة ملاحظات تفصيلية
- عرض شبكي مع عوامل التصفية
- تغيير الحالة بسهولة

### 6. الاستهلاك والاهلاك (Consumption) ⭐ **جديد**
**تسجيل عمليات الاستهلاك والخسارة:**
- تسجيل الكميات المستهلكة أو التالفة
- أسباب مختلفة:
  - تلف وفساد
  - استخدام في المتجر
  - عينة مجانية
  - رصيد جرد
  - خسارة
  - أخرى
- تحديث تلقائي للمخزون
- تتبع شامل لكل عملية استهلاك
- عرض شبكي قابل للبحث
- ملاحظات إضافية اختيارية

### 7. الفواتير (Invoices)
**إدارة شاملة للفواتير:**
- عرض جميع الفواتير المسجلة
- تفاصيل كاملة لكل فاتورة
- البحث والتصفية
- تصدير إلى CSV
- طباعة الفاتورة
- حالة الدفع

### 8. التقارير (Reports)
**تحليل الأداء والإحصائيات:**
- رسوم بيانية تفاعلية
- الإيرادات اليومية
- المصروفات والأرباح
- عدد العمليات اليومية
- تقارير شاملة حسب التاريخ

## 🎨 نظام العرض الشبكي (Grid System)

جميع الصفحات الجديدة تستخدم نظام عرض شبكي احترافي:

```
Grid Layout:
- شاشات صغيرة (Mobile): 1 عمود
- شاشات متوسطة (Tablet): 2 عمود
- شاشات كبيرة (Desktop): 3 أعمدة

Bootstrap Grid Breakpoints:
- md: 768px (2 عمود)
- lg: 1024px (3 أعمود)
```

### مميزات الشبكة:
✓ تصميم مستجيب (Responsive)
✓ تأثيرات التمرير (Hover Effects)
✓ بطاقات مع ظلال زجاجية (Glassmorphism)
✓ تنسيق محاذي (Balanced Spacing)
✓ ألوان متناسقة

## 💾 إدارة البيانات

### تخزين البيانات:
- **LocalStorage**: تخزين محلي في المتصفح
  - `pos_products`: المنتجات
  - `pos_purchases`: المشتريات
  - `pos_productions`: الإنتاج
  - `pos_consumptions`: الاستهلاك
  - `pos_invoices`: الفواتير
  - `pos_reports`: التقارير اليومية

### الحفظ التلقائي:
- تتم حفظ جميع البيانات تلقائياً عند التغيير
- لا فقدان للبيانات عند إغلاق المتصفح
- يمكن استرجاع البيانات من أي متصفح/جهاز نفسه

## 📋 تفاصيل كل صفحة

### صفحة المشتريات:
```
المدخلات:
- المورد (نص)
- المنتجات (اختيار من القائمة)
- الكمية لكل منتج (رقم)
- السعر للوحدة (رقم عشري)
- الملاحظات (نص اختياري)

المخرجات:
- تحديث تلقائي لكمية المخزون
- تسجيل سعر الشراء للمنتج
- حفظ في قائمة المشتريات
```

### صفحة الإنتاج:
```
المدخلات:
- المنتج (اختيار من القائمة)
- كمية الإنتاج (رقم)
- الملاحظات (نص اختياري)

الحالات:
- قيد الانتظار (Pending) - ⏳
- مكتمل (Completed) - ✅
- ملغى (Cancelled) - ❌

المخرجات:
- تحديث تلقائي للمخزون عند الإتمام
- تتبع شامل لعمليات الإنتاج
```

### صفحة الاستهلاك:
```
المدخلات:
- المنتج (اختيار من القائمة)
- الكمية المستهلكة (رقم)
- السبب (اختيار من قائمة محددة)
- الملاحظات (نص اختياري)

المخرجات:
- تقليل المخزون تلقائياً
- تسجيل سبب الاستهلاك
- حفظ في قائمة الاستهلاك
```

---

## 🔄 تدفق العمليات

### دورة البيع:
```
1. شراء من المورد → المشتريات
2. استقبال المنتج → تحديث المخزون
3. بيع للعميل → نقطة البيع
4. إنشاء فاتورة → الفواتير
5. تحليل الأداء → التقارير
```

### دورة الإنتاج:
```
1. تسجيل عملية إنتاج → الإنتاج
2. استقبال المنتجات → تحديث المخزون
3. بيع المنتجات → نقطة البيع
4. تحليل الإنتاجية → التقارير
```

### إدارة المخزون:
```
1. الإدخال:
   - شراء (من المشتريات)
   - إنتاج (من الإنتاج)

2. الإخراج:
   - بيع (من نقطة البيع)
   - استهلاك (من الاستهلاك)

3. النتيجة:
   - رصيد محدّث دائماً
   - تنبيهات عند النقص
```

## 🚀 نصائح للاستخدام الأمثل

1. **ابدأ بإضافة المنتجات** في صفحة المخزون
2. **سجّل المشتريات** عند استقبال منتجات من المورد
3. **تتبع الإنتاج** إن كنت تنتج منتجات
4. **سجّل الاستهلاك** للكميات التالفة أو المستخدمة
5. **استخدم نقطة البيع** للبيع السريع
6. **راجع التقارير** بشكل دوري لتحليل الأداء
