# تحديث نظام Excel والجنيه المصري

## ما الجديد

تم إضافة ميزات جديدة وحسّابات مالية بالجنيه المصري:

### 1. نظام Excel المتكامل
- تصدير جميع البيانات إلى ملف Excel احترافي
- استيراد البيانات من ملفات Excel
- تنظيم البيانات في 5 ورقات منفصلة

#### الأوراق المصدّرة:
```
1. المنتجات
   - رقم المنتج
   - اسم المنتج
   - السعر (ج.م)
   - سعر الشراء (ج.م)
   - الفئة
   - الكمية
   - الباركود

2. الفواتير
   - رقم الفاتورة
   - التاريخ
   - عدد المنتجات
   - الإجمالي الجزئي (ج.م)
   - الضريبة (ج.م)
   - الإجمالي (ج.م)
   - الحالة (مدفوعة/غير مدفوعة)
   - طريقة الدفع
   - اسم العميل

3. المشتريات
   - رقم الشراء
   - التاريخ
   - المورد
   - عدد المنتجات
   - الإجمالي (ج.م)
   - الملاحظات

4. الإنتاج
   - رقم الإنتاج
   - التاريخ
   - اسم المنتج
   - الكمية المنتجة
   - الحالة
   - الملاحظات

5. الاستهلاك
   - رقم الاستهلاك
   - التاريخ
   - اسم المنتج
   - الكمية
   - السبب
   - الملاحظات
```

### 2. الحساب بالجنيه المصري
- العملة: الجنيه المصري (EGP)
- الرمز: ج.م
- ضريبة القيمة المضافة: 14%
- جميع الحسابات المالية بالعملة الجديدة

#### التطبيق على جميع العمليات:
- المبيعات والفواتير
- الشراء والمشتريات
- التقارير والإحصائيات

### 3. صفحة الإعدادات الجديدة

- خيار آمن لحذف جميع البيانات

## كيفية الاستخدام

### تصدير البيانات إلى Excel
1. اذهب إلى صفحة "اكسل"
2. اضغط على "تصدير جميع البيانات إلى Excel"
3. سيتم تحميل الملف تلقائياً

### استيراد من Excel
1. اذهب إلى صفحة "اكسل"
2. اضغط على "استيراد من Excel"
3. اختر ملف Excel
4. سيتم استيراد البيانات تلقائياً

### عرض الإعدادات
1. اذهب إلى صفحة "الإعدادات"
2. شاهد معلومات العملة والنظام
3. عرض إحصائيات البيانات الحية

### حذف جميع البيانات
1. اذهب إلى صفحة "الإعدادات"
2. انزل إلى قسم "حذف جميع البيانات"
3. اضغط على "حذف جميع البيانات"
4. أكد الحذف في النافذة الظاهرة
